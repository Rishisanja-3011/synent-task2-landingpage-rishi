## 🌟 About

A fully responsive, luxury fashion landing page for a fictional brand called **AURA**. Built with pure HTML, CSS, and JavaScript — no frameworks.

---

## ✅ Features

- **Hero Section** — editorial layout with animated product cards & live stats
- **Marquee Banner** — animated scrolling text strip
- **Collections Grid** — 3D tilt effect on hover with overlay details
- **Philosophy Section** — dark-theme feature cards
- **Testimonials** — three-column layout with featured center card
- **CTA Banner** — urgency-driven section with decorative circles
- **Footer** — full 4-column layout with social links
- **Responsive Navbar** — sticky with scroll effect + hamburger menu for mobile
- **Custom Cursor** — gold dot + trailing ring
- **Scroll Reveal** — staggered fade-up animations on all sections
- **Smooth Scroll** — all anchor links scroll smoothly

---

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| Desktop (>1024px) | Full 2-column hero, side-by-side sections |
| Tablet (768–1024px) | Single column hero, stacked features |
| Mobile (<768px) | Hamburger nav, stacked grid, full-width sections |
